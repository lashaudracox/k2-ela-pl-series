# K–2 ELA Professional Learning Series — Ross Elementary
Static site for the Q1 PL series (7 sessions + participant guides + classroom materials).

## Deploy on GitHub Pages
1. Create a new repo (e.g. `k2-ela-pl-series`) — Public.
2. Upload the entire contents of this folder (drag and drop works on github.com → "Add file" → "Upload files").
3. Add the five brand images to the `assets/` folder (see assets/PUT-IMAGES-HERE.txt).
4. Repo → Settings → Pages → Source: "Deploy from a branch" → Branch: `main`, folder `/ (root)` → Save.
5. After ~1 minute the site is live at `https://YOUR-USERNAME.github.io/k2-ela-pl-series/`.

`index.html` is the Series Hub — that URL is the only link teachers need.
