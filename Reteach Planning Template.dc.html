Reteach Planning Template

A small-group reteach supplies a missing access point — not a slower repeat of the same lesson.
Teacher: ____________________
Grade: ______
Source lesson (unit/wk/day): ______________
Reteach date / slot: ______________

1 • The Evidence

The need, named precisely — from the sorted work (tricky words / digraphs / complete sentences / …)
The group — students grouped by need, not level (names)
What the work shows — 2–3 specific examples from the papers

2 • The Access Point

The move — circle one:  visual aid  •  student discourse  •  model / think-aloud  •  prior knowledge
The plan — what I will make, say, and have students do (the grade-level task stays)
Every-voice check — how each student in the group practices, not watches (boards, choral, tapping, Turn & Talk)
ESL / SPED partner input — language vs decoding need, IEP connections

3 • After the Reteach

The check — the quick task that shows whether the need is met
Who moved piles — names + evidence: ____ of ____ now mastered / approaching
Next move — fade the scaffold, run a second round, or hand off to RTI² with this evidence
The test: if the reteach could be described as "the same lesson, slower," go back to step 2 — name the access point that was missing the first time and supply exactly that.
