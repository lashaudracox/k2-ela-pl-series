K–2 ELA PROFESSIONAL LEARNING • QUARTER 1 • FACILITATOR AGENDA

Mastering Foundational Skills for ALL Students

LaShaudra Cox • Ross Elementary • Biweekly, 60 minutes, inside K–2 ELA planning • August–October
# Date Session focus Facilitator prep Teachers bring / leave with
S1 Inservice week Building a culture of learning — access from Day One Print guides; queue observation snapshot data (11/18, 14/18); chart paper for routine menu Leave with: exit ticket — one participation routine + evidence of access
S2 Thu, Aug 13 Intellectual preparation — annotating lessons with access points Confirm ESL/SPED coverage; collect S1 exit tickets; flag this week's lessons in each grade's Wonders map Bring: curriculum map + taught routine. Leave with: one annotated lesson, taught before S3
S3 Thu, Aug 27 Core Action 3 — ALL students listening, speaking, reading, writing Print IPG Core Action 3 indicators A–C; whiteboards/hand-signal demo materials; decodable text sets Bring: taught annotation + count. Leave with: participation counts from one self-selected activity
S4 Thu, Sep 10 The scaffolding toolkit — temporary, student-specific supports Confirm ESL/SPED coverage; sample anchor charts, sentence starters, modeled-writing examples Bring: student work + counts. Leave with: one scaffold planned from student-work data
S5 Thu, Sep 24 Analyzing student work through the lens of access Confirm ESL/SPED coverage; sorting protocol mats (mastered / approaching / needs access point) Bring: a class set of student work. Leave with: sorted set + one planned reteach access point
S6 Thu, Oct 8 Feedback that builds coherence — peer observation frames Schedule peer observation windows or collect video clips; print "Because you …, your students were able to …" frames Leave with: one completed peer-feedback frame from a walkthrough or video
S7 Thu, Oct 22 Data review — did it work? — and the Quarter 2 action plan Pull quarter data: participation counts, mastery rates, universal screening / RTI²; celebration plan Leave with: team data summary + Quarter 2 commitments
Every session, same shape
Open with last session's evidence (10 min) → new learning (15–20) → working time in the guides (20–25) → exit-ticket commitment (5–10).
Standing logistics
ESL & SPED partners join S2, S4, S5. Teachers always bring their grade-level Wonders map. Dates flex with the school calendar — keep the sequence.
Through-line
What we plan together is what we teach, what walkthroughs look for, what feedback names, and what student work measures.
